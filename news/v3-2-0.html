Announcement: Kickstarting the 4th Generation (V4.x.x)

 We are thrilled to announce that work has officially begun on the new version of the script, marking the transition to the Fourth Generation (V4.x.x). This upcoming release is not merely a minor update; the script has been completely rebuilt from the ground up while maintaining full compatibility with existing databases for current users. Powered by Laravel 12, this version prioritizes enhanced security and protection. It is also designed to be significantly more developer-friendly, streamlining the process for developers and designers to create plugins, themes, and further customize the script. We look forward to receiving your suggestions and feedback for this upcoming release.
